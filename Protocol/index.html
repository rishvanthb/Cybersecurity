Internet Protocols

Internet protocols are the rules that allow computers and other digital devices to communicate with each other over the internet. These protocols define how all different kinds of data are transmitted, and recieved, making sure that allow information is transported and arrives at its destination correctly.


Some important protocols include:

  • HTTPS (Hypertext Transfer Protocol): Helps with loading text and images in webpages(such as this scavenger hunt)

  • TCP/IP (Transmission Control Protocol/Internet Protocol): Manage how data is transmitted between devices by splitting it up into many different "packets" which are dissassembled once sent and reassembled upon recieval.

  • DNS (Domain Name System); Translates website names into IP addresses that computers can understand
